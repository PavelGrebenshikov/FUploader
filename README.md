# FUploader

## Uploading the file to the server and processing.

### Download FUploader.
```
git clone https://github.com/PavelGrebenshikov/FUploader.git
```
```
cd FUploader/
```
### Installation.
```
pip install -r requirements.txt
```
```
cd FUploader/
```
```
python3 manage.py runserver
```
#### or
```
python manage.py runserver
```
